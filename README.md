# 🦇 Bat RPG - Night Survivor

A kid's Bat game we built together to explore **agentic coding** — building a working game entirely through conversation with an AI coding assistant, no manual coding required.

Computer Science Education is a passion of mine so I am posting this here so we can watch and share as the kids' games evolves.

Play as a bat surviving waves of night creatures. Auto-attack enemies, collect XP and gold, level up, and spend skill points on RPG-style stats (Vitality, Strength, Agility, Focus, Echo Power) to grow stronger and push further into the night.

## How to Play

- **WASD** or **Arrow Keys** — Fly around
- **Auto-attack** — Your bat automatically bites the nearest enemy in range
- **E** — Echo Blast, an area-of-effect ultimate attack (has a cooldown)
- **U** — Open the Upgrade menu to spend Skill Points
- **H** — Open the Help screen (controls reminder, in-game)

Survive as many waves as you can. Your best wave is saved in the browser so you can try to beat it next time.

## Running it

No install or build step needed — it's plain HTML/CSS/JavaScript.

1. Download or clone this repository
2. Open `index.html` in any modern web browser

## About

This project was created as a fun example of **vibe coding**: describing the game you want in plain language and having an AI assistant (GitHub Copilot) write, test, and refine all the code.
